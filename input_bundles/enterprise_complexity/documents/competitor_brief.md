# Competitive Brief — EnterpriseSuite

## Competitor A: PlatformOne
- **Strengths:** Best-in-class enterprise features: SSO, SCIM provisioning, custom RBAC, audit logging, data residency. SOC 2 Type II and ISO 27001 certified. 8 regional data centers.
- **Weaknesses:** Very expensive ($85/seat/month enterprise tier), slow release cadence (quarterly), UI is functional but dated.
- **Recent moves:** Added FedRAMP certification in Q3 2025 targeting government contracts. Raised Series D at $2B valuation.

## Competitor B: WorkLayer
- **Strengths:** Modern UI, competitive pricing ($25/seat/month), strong API ecosystem with 200+ integrations. Fast onboarding (average 5 days).
- **Weaknesses:** No data residency controls, basic RBAC only, SOC 2 Type I only (not Type II). Limited to US and EU regions.
- **Recent moves:** Launched AI assistant feature in Q4 2025. Growing rapidly in mid-market but losing enterprise deals to PlatformOne on compliance requirements.
